# 다음 장면 Board Game Prototype

보드게임 `다음 장면`을 실제 제작하기 전에 게임성, 구성물, 카드 디자인, 진행 흐름을 검토하기 위한 Next.js 프로토타입입니다.

## What It Shows

- 게임이 실제로 성립하는지에 대한 구조 진단
- 카드, 개인 인생판, 시간 토큰, 삶의 방향 카드 디자인 제안
- 24장 인생 카드 전체 브라우저와 가치/종류 필터
- 한 라운드 진행을 시각적으로 따라가는 애니메이션 데모
- 첫 테스트에서 확인할 항목과 밸런스 수정 레버

## Stack

- Next.js
- React
- Framer Motion
- Lucide React
- Codex-generated SVG mockup assets

## Local

```powershell
npm install
npm run dev
```

## Vercel

- Build command: `npm run build`
- Framework: Next.js
